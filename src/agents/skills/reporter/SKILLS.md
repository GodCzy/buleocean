# SQL Reporter Skill

该技能用于在需要数据库报表时，为代理提供一套统一的查询与报告工作方式。

## 适用场景

- 统计业务数据
- 生成汇总报表
- 查询指标变化
- 输出图表化结论

## 工作流程

1. 理解用户的报表目标
2. 生成正确且高效的 SQL 查询
3. 执行查询并获取结果
4. 使用图表工具生成可视化结果
5. 输出最终报告，而不是只输出原始 SQL

## 约束

- SQL 必须可读、正确、尽量高效
- 避免无必要的全表扫描
- 报告应以结论为中心
- 若生成图表，最终结果需要把图表以 Markdown 图片形式嵌入说明中

## 可用能力

- 数据库查询工具
- 图表生成工具
- 必要时使用检索工具补充上下文
